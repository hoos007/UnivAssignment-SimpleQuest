안드로이드 일정 관리 어플

이 프로젝트는 대학교 3학년 2학기 모바일 프로그래밍 과목의 기말평가로 진행한 프로젝트 입니다.
## 프로젝트 개요
### 이름
심플 퀘스트

### 기획 배경
직접 일정관리 앱을 개발하고 싶어서 진행한 프로젝트입니다. 일정 관리 앱을 개발하면서 여러 레이아웃을 구상하고 DB를 사용해보는 경험을 통해 많이 배울 수 있겠다는 생각 때문에 이 아이디어를 채택했습니다.

### 주요 기능
1. 일정 저장기능
2. 중요한 일정 체크 기능
3. TODAY페이지
4. 캘린더 기능
